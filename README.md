📌Project Overview

This project focuses on analyzing pizza sales data using Power BI to uncover business insights and visualize important KPIs through an interactive dashboard.

The dashboard helps understand:

Sales performance
Customer ordering behavior
Product performance
Revenue trends
Best and worst-selling pizzas

🎯 Problem Statement

The objective of this project is to analyze pizza sales data and create an interactive dashboard to monitor:

Revenue performance
Order trends
Customer preferences
Sales contribution by pizza category and size
Top and bottom-performing pizzas

📊 KPIs Implemented
Total Revenue
Average Order Value
Total Orders
Total Pizzas Sold
Average Pizzas Per Order

📈 Charts & Visualizations
1. Daily Trend for Total Orders
Bar chart showing daily order patterns
2. Monthly Trend for Total Orders
Line chart displaying order trends over time
3. Percentage of Sales by Pizza Category
Pie chart representing category-wise sales contribution
4. Percentage of Sales by Pizza Size
Pie chart showing customer preference by pizza size
5. Total Pizzas Sold by Category
Funnel chart comparing category-wise sales
6. Top 5 Best Sellers
Based on Revenue, Quantity, and Orders
7. Bottom 5 Worst Sellers
Identified low-performing pizzas
🛠 Technologies Used
Power BI
DAX
Power Query
Excel / CSV Dataset

🔍 Key Insights
Certain pizza categories generated the highest sales contribution
Peak ordering activity was observed during specific days and time periods
Large-sized pizzas contributed significantly to total revenue
Best-selling pizzas consistently drove business performance
Low-performing pizzas highlighted areas for menu optimization

Data Cleaning
